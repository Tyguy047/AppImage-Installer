# AppImage Installer

## About
**App Image Installer** helps you install .AppImages to your Linux system so they can be searched for in the start menu.

## Some notes and features:
- Developed using Pytohn 3.14
- Features a Tkinter GUI
- As of this commit, **AppImage Installer** is not yet usable and is still under development!
- **AppImage Installer**  is being developed on `Ubuntu 24.04.3 LTS x86_64`
